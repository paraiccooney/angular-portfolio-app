Languages - HTML, CSS.

Additional technologies - Bootstrap, Hover.css

My first portfolio entry is a collection of static web-pages designed to display information of a tourist attraction in the South-East. It is built solely using HTML & CSS & also utilises Bootstrap for it's layout & styling. Finally it encorporates the Hover.css library in order to add a level of animation to the buttons & increase the overall user experience.

Repo

Deployed

Navbar - allows the user to navigate to any section of the site from any page & is visible in the fold upon landing to reduce page-bounces.

Footer animation - this will provide the user with a pleasant & playful user experience.

Embedded maps - there a a number of maps integrated onto the site to allow the user to immediately view where each bike rental resource is & view the entire route in detail.

Responsive design - The site was designed using a mobile first approach. Both the navbar & informational/functional divs expand on larger screens to utilise the full screen of the user & display information more concisely with a greater proportion of elements being displayed above the fold.

Border timeline - A border timeline has been included & is visible on larger screen sizes on the Places Of Interest page to visually demonstrate to the user that the locations being described are in the order that they will be experienced by the user in real life should they be travelling one direction.

Contact - when fully connected this allows the user to contact the administrator for more information. This will help the administrator to review & determine if additional information/features are to be included in future.

Gallery - In future I would like to implement a gallery section where users could upload images which the site administrator could then "pin" to the top of the page.

Facilities Page - In future a features page could be implemented to provide additional information. To date this information is not available. The features page was included in the original wireframes however all the information is not available as of yet (toilets, parking, child/elderly/disable friendly, & water stations) so this is currently omitted.

Email.js - prior to launch the contact section will need to be wired up. To achieve this I plan to use the email.js library.

Languages - HTML, CSS, JavaScript.

Additional technologies - Bootstrap, jQuery, Jasmine

To demonstrate my use of JavaScript logic I developed an in browser version of the 1980s game Simon. The game takes place on one webpage & jQuery is used to select elements & update the HTML with new score & roll values. Additionally this project makes use of the latest HTML capabilities with the use of the HTML5 audio tag which is used to improve user experience by adding sound to button clicks.
This project also makes use of the Jasmine testing library to ensure high quality code.
