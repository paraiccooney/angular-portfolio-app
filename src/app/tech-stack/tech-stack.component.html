Hypertext Markup Language - the cornerstone of the internet. Like many developers my first introduction to hard-coding was with HTML & although I have studied many languages & frameworks since I have always managed to keep a solid grounding in the basics. Good HTML conventions can make the code a lot more readable not only for the coder who created it but also for other developers & collaborators. I employ these conventions & am familiar with the latest version of HTML (HTML5) & it's capabilities.


CSS or cascading style sheets are the styling behind the internet. They're also responsible for a large number of animations that give the internet & applications their interactivity & intuative feel. CSS compliments the practicality of HTML. I employ CSS to give my websites & applications enticing & intuative interfaces. I write my own CSS to do so while also implimenting some useful libraries such as Bootstrap & Hover.css (among others).


Bootstrap is an open source library built using HTML, CSS , & JavaScript. It contains a range of pre-generated code that includes styling & behaviour. Bootstrap has enabled me to utilise the power of JavaScript without actually writing any JavaScript. What's more Bootstrap has enabled me to style & structure my sites & applications more efficiently by simply binding Bootstraps pre-written classes to my elements. One of the most powerful & popular examples of this power is Bootstraps grid system.
A popular extension of Bootstrap is Bootswatch - a website built using Bootstrap fundamentals but which goes further in providing a number of ready-built themes. One of these themes (Lux) can be seen in my Python with Django project.
Bootstrap is a very easy to use yet powerful library which I have incorporated into most of my projects.


jQuery is a very popular JavaScript library used to increase the ease & efficiency of selecting DOM elements for manipulation. It is lightweight yet powerful & something I have used in many projects to make my code more readable & efficient.


JavaScript is one of the best loved programming languages out there. If HTML is the noun & CSS the adjective of the internet then JavaScript is the verb. JavaScript has powerful logic abilities & I have used this reasoning in many of my projects. JavaScript also has many supporting libraries & APIs. For example the Stripe payment API can be implemented & interacted with using JavaScript.
It is also worth mentioning TypeScript at this point. TypeScript is a superset of JavaScript meaning that all relevant JavaScript is interpretable using TypeScript. TypeScript builds on top of JavaScript & has a number of additional functionalities & more concise ways of writing code. TypeScript is something I also utilise in my development & this application has been built using TypeScript.


Application Programming Interfaces or APIs are the interfaces between our devices & servers. They allow us to communicate with the servers & return information to us based on our request. There are many pre-built, open, & free APIs available which allow us to interact with back-end servers & return information.
I am familiar with these open APIs & have interacted with them from the front-end & seen a number of walkthrough projects throughout my development.
I plan to build an API of my own as a project in future.


Python is one of the world's most popular & supported programming languages. Python is a soft-typed language & is known for it's vast array of importable packages that assist users in a wide array of tasks from web-development to machine learning & artificial intelligence.
I have used Python when developing a number of applications (please see "Python with Flask" & "Python with Django" in the portfolio section) & have found it to be very intuitive & powerful.


Flask is a lightweight Python microframework used to build dynamic applications. It is lightweight yet powerful. I have employed Flasks capabilities when building my recipe blog application (please see the portfolio section). Flask makes use of the Jinja templating languages which enabled me to inject logic directly into my html - cutting down on the size of my code & increasing readability. Flask also facilitates routing.


Django is one of the world's most popular Python frameworks. Unlike Flask it is a more complete framework & has much more power under the hood. Django also makes use of routing & Jinja templating logic. However it goes further with many additional functionalities included built in authorisation, admin panel, & back-end capabilities.
Using Django I developed my Newspaper application complete with dynamic rendering, back-end database, authentication, & authorisation.


Structure Query Language or SQL is a common programming language used to interact & return information from relational databases. It is the industry standard in many respects.
SQL is something I am familiar with & was exposed to during my Full-Stack course with The Code Institute. I am familiar with running SQL scripts in the command line & also using sql files.


Stripe is a payment technology used to process payments online in a secure manner. It is increasingly popular & used in a range of ecommerce sites.
I have implemented Stripes JavaScript API on my Newspaper application (please see "Python with Django" in the Portfolio section).


MongoDB is an online database resource which can serve as the back-end of our applications/websites. Mongo structures its database in a similar manner to JSON (JavaScript Object Notation) files making them easy to query & interact with as this is a very common way of structuring data.
I have used MongoDB to structure & store the data for my Recipe-Blog Application (please see "Django with Flask" in the Portfolio section).


Amazon Web Services or AWS refers to an offering of web services by Amazon. Services range to cover a broad array of requirements.
I have used Cloud9 as my IDE (Cloud9 was taken over by AWS in 2019) & have also used their authorisation service (IAM) & S3 services (AWS Buckets) to host data.


Angular is a component based front-end framework with a lot of power. Angular was originally written using JavaScript & was referred to as AngularJS, however more recent versions are now written in TypeScript (please refer to the JavaScript section for more information).
Angular enables developers to write fewer lines of code through templating & the building of reusable components. It also facilitates animations, event listening, & routing.
Angular is a popular method of developing single page applications (such as the one you are presently using) which allow for DOM elements to be destroyed & created without the need to reload/download the entire DOM tree.
