Hypertext Markup Language - the cornerstone of the internet. Like many developers my first introduction to hard-coding was with HTML & although I have studied many languages & frameworks since I have always managed to keep a solid grounding in the basics. Good HTML conventions can make the code a lot more readable not only for the coder who created it but also for other developers & collaborators. I employ these conventions & am familiar with the latest version of HTML (HTML5) & it's capabilities.

CSS or cascading style sheets are the styling behind the internet. They're also responsible for a large number of animations that give the internet & applications their interactivity & intuative feel. CSS compliments the practicality of HTML. I employ CSS to give my websites & applications enticing & intuative interfaces. I write my own CSS to do so while also implimenting some useful libraries such as Bootstrap & Hover.css (among others).
